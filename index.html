Sequential Teams
by Aditya Mahajan

A graphical modeling approach to simplifying sequential teams

Introduction

This is a Haskell library that implements the algorithm presented in

Aditya Mahajan and Sekhar Tatikonda, " A graphical modeling approach to simplifying sequential teams," proceedings of 7th International Symposium on Modeling and Optimization in Mobile, Ad Hoc, and Wireless Networks (WiOpt), Control over Communication Channels (ConCom) Workshop, Seoul, South Korea, June 27, 2009.

Installation

Download either the zip or the tar formats. Copy the file to an empty directory and uncompress. This project depends on fgl and graphviz libraries.

Install the package either using cabal install or manually.

  • Installing using cabal install. 
    $ cabal install teams
  • Installing manually. 
    $ runhaskell Setup configure
$ runhaskell Setup build
$ runhaskell Setup install

You can also clone the project with Git by running: 

$ git clone git://github.com/adityam/teams

Example

Lets try to derive the structural results for the real-time source coding problem considered in

Hans S. Witsenhausen, "On the structure of real-time source coders," Bell Systems Technical Journal, vol 58, no 6, pp 1437-1451, July-August 1979.

We use a EDSL to specify the system. 

x  = mkNonReward "x"
x' = mkNonReward "x'"
y  = mkNonReward "y"
m  = mkNonReward "m"
r  = mkReward    "r"

f = mkDynamics "f"
c = mkControl  "c"
g = mkControl  "g"
l = mkControl  "l"
d = mkDynamics "d"

dynamics t | t == 1 = f(1).$.(x(1) .|. [])
                   ++ c(1).$.(y(1) .|. [x(1)])
                   ++ g(1).$.(x'(1).|. [y(1)])
                   ++ l(1).$.(m(1) .|. [y(1)])
                   ++ d(1).$.(r(1) .|. [x(1), x'(1)])
           | otherwise = f(t).$.(x(t) .|. [x(t-1)])
                      ++ c(t).$.(y(t) .|. map x[1..t] ++ map y[1..t-1])
                      ++ g(t).$.(x'(t).|. [y(t), m(t-1)])
                      ++ l(t).$.(m(t) .|. [y(t), m(t-1)])
                      ++ d(t).$.(r(t) .|. [x(t), x'(t)])

rt = mkTeamTime dynamics 4

Load this file in ghci and run 

*Main> printTeam rt
