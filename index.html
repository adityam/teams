Simplifying Sequential Teams
by Aditya Mahajan

A graphical modeling approach to simplifying sequential teams

Installation

You can download this project in either zip or tar formats. Copy the file to an empty directory and uncompress. This project depends on fgl and graphviz libraries. If you have 

cabal install
on your system, you can run 
cabal install teams
You can also install this package by 
$ runhaskell Setup configure
           $ runhaskell Setup build
           $ runhaskell Setup install

You can also clone the project with Git by running: 

$ git clone git://github.com/adityam/teams
